About The Event

Got Game is a student led game jam where clubs such as Artificial Intelligence Society (AIS), Student Game Developer Alliance (SGDA), Extended Reality Society (XRS), and Makerspace come together to create an event where students can help each other in learning a new skill or showcasing a talent.

Where

UT Dallas Makerspace.
3000 Waterview Pkwy, Richardson, TX

When

Saturday 9am - Sunday 2pm
March 30th - 31st


Tracks
