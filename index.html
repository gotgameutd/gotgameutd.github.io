About The Event

Got Game is a student led game jam where clubs such as Artificial Intelligence Society (AIS), Student Game Developer Alliance (SGDA), Extended Reality Society (XRS), and Makerspace come together to create an event where students can help each other in learning a new skill or showcasing a talent.

F.A.Q

  • What is Got Game?

    Got Game is a student led game jam where clubs such as Artificial Intelligence Society (AIS), Student Game Developer Alliance (SGDA), Extended Reality Society (XRS), and Makerspace come together to create an event where students can help each other in learning a new skill or showcasing a talent.

  • Who can participate?

    All students are welcome. This includes undergraduate students and graduate students.

  • How big are teams?

    Teams are usually composed of 2-4 people. No more than 4 people on a team. You can do it alone, but it'll be less fun.

  • What if I don't have teammates?

    Weâ€™ll have a scheduled time for team creation after the opening ceremony.

  • What should I bring?

    Laptops, chargers, cell phones, and of course passion are all helpful. Other useful items include toiletries, a change of clothes, a pillow, some earplugs, an eye mask, a jacket, and... deodorant. Knives, firearms, and other dangerous things are not permitted at the event.

  • How much does this event cost?

    Got Game is 100% free for students. This includes WiFi, food, drinks, and miscellaneous swag for the entire jam.

  • Are we allowed to build on past projects?

    A Game Jam is an experience to build something new within a set amount of time. Therefore, you cannot work on past projects. However, using your own open-source or third-party libraries is fine, as long as the project is new.

  • Will Got Game provide reimbursements?

    Sadly Got Game is not providing reimbursements for travel!

  • What is the theme of Got Game?

    We will be announcing the theme of the game jam during the opening ceremony so stay tuned!
